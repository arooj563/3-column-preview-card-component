# 3-Column Preview Card Component

A clean, responsive, and accessible **3-column preview card component** built with modern **HTML5** and **CSS3**.  
Designed for product showcases, feature highlights, marketing cards, and UI component libraries.

---

## 📌 Overview

This component presents three structured content cards with consistent styling and subtle hover interactions.  
It is fully responsive and optimized for mobile, tablet, and desktop screen sizes.

---

## ✨ Key Features

- **Modern, minimal UI design**
- **Fully responsive layout** (Flexbox/Grid based)
- **Accessible typography and color contrast**
- **Lightweight & dependency-free**
- **Easily customizable styles and colors**
- **Reusable component for multiple use cases**

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3 (Flexbox + Grid)**
- **Google Fonts** (optional)
- **SVG icons** (optional)

---

## 📱 Responsive Design

- **Desktop:** Three cards aligned horizontally  
- **Tablet:** Two columns layout  
- **Mobile:** Single column stack  
- Auto-adjusted padding, font sizes, and spacing
